# 更新日志 — 2026-05-19

## 1. 用户信息显示

- **主播端/观众端** 底部/侧边栏显示当前登录用户的昵称和手机号
- 信息通过环境变量 `LIVE_NAME`、`LIVE_TEL` 从 Launcher 传递给子进程

## 2. 观众端按钮布局重构

重新组织播放器底部按钮栏，更清晰的分工：

- **自定义服务**（合并原"打开文件"+"URL"+"推流服务"）
  - 下拉菜单：打开文件 / RTMP 直播推流 / RTMP 点播 / HLS 点播 / 自定义URL
- **网络直播** 独立按钮，点击直接弹出频道列表
- **在线主播**（原"房间"按钮）拉取正在直播的房间

## 3. 网络直播频道列表

- 解析 `url.txt` 获取频道名和 M3U8 播放地址
- 来源：GitHub 开源项目 `best-fan/iptv-sources` 每日自动更新
- 共 70 个频道：CCTV-1~CCTV-17、省级卫视、黑莓系列
- 支持 UTF-8 中文频道名显示

## 4. 房间列表优化

- 添加 Cancel 按钮，用户可不选房间直接关闭对话框
- 通过"在线主播"按钮可随时重新打开房间列表
